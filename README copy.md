# (NaFR) — Blues & Greens with a Touch of Sludge

Website for **Not a Fishing Rod (NaFR)**, a small instrumental music project.

## About

Not a Fishing Rod (NaFR) is a musical project built from fragments: short guitar sketches captured in quiet moments and later brought into the studio to be expanded into full pieces. The music lives somewhere between memory and improvisation.

**Blues & Greens with a Touch of Sludge** gathers instrumental sketches written between 2018 and 2025.

## Structure

Single-page site built on the [Hyperspace](https://html5up.net/hyperspace) template by HTML5 UP.

```
nafr-music/         ← site root
├── index.html
├── assets/
│   ├── css/
│   ├── js/
│   ├── sass/
│   └── webfonts/
└── images/
    ├── nafr-logo.png
    └── album-cover.jpg
```

## Deployment

No build step required. Open `nafr-music/index.html` locally or serve the `nafr-music/` directory from any static host (Netlify, GitHub Pages, etc.).
